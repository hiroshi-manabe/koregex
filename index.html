Syntax

Hangul Block

{초중종} describes one Hangul syllable. Outside braces, regex is preserved.

{ㅎㅏㄴ}
exactly 한
{..ㄴ}
any initial, any vowel, final ㄴ
{하.?}
하, 학, 한, ...

Wildcards

. means any value for that slot. In final position, it means any non-empty final.

{ㅎ.}
ㅎ + any vowel + no final
{ㅎ..}
ㅎ + any vowel + any final
{ㅎ..?}
ㅎ + any vowel, with or without final

Composition

Use inner braces to compose one slot value.

{{ㄱㄱ}.}
initial ㄲ + any vowel + no final
{.{ㅗㅏ}}
any initial + vowel ㅘ + no final
{..{ㄹㄱ}}
any initial + any vowel + final ㄺ

Classes

Classes are slot-local. ^ negates the listed values.

{[ㄱㄴㄷ]ㅏㄴ}
initial ㄱ, ㄴ, or ㄷ
{ㄱㅏ[^ㄴ]?}
no final, or a final other than ㄴ
{ㄱㅏ[ㄴ{ㄹㄱ}]}
final ㄴ or ㄺ
